About

Brite is a Responsive HTML5 / CSS3 simple, clean and Professional theme. It comes with 10+ Unique layouts, Many more inner Pages, Awesome Slideshows, Unique Color Variations. It's perfect for any business like business, hosting, restaurant, portfolio, agency and many more...

How to use

Upload "html" folder into your ftp server. This folder contains all files of the template.

  • images
